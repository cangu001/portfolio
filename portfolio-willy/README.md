## Déploiement sur Netlify

### Option 1 — Via Git (recommandé)
1. Push ce repo sur GitHub
2. Sur [netlify.com](https://netlify.com) → *Add new site* → *Import from Git*
3. Sélectionne ton repo → Deploy
4. Le formulaire de contact fonctionne automatiquement

### Option 2 — Drag & Drop
1. Sur [netlify.com](https://netlify.com) → *Deploy manually*
2. Glisse le dossier → Deploy en 30 secondes

## Personnalisation

- **GitHub link** : Recherche `https://github.com/` dans `index.html` et remplace par ton vrai lien
- **Projets** : Cherche "Projekt 2" et "Projekt 3" pour compléter tes projets
- **Photo / CV** : Déjà intégrés
